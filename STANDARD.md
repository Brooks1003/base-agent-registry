# Agent Registry Standard v4.0 — 最终版

## 摘要

AI Agent在区块链上的身份、能力、信誉、质押、审计和追责标准。
Agent不存在"失误"——只有代码正确和代码错误。
创建者不需要懂代码——审计者替你看。
**罚没金没人能碰——50%给举报人，50%永远销毁。**

---

## 核心原则

### 第一原则：Agent的信誉是"代码可信度"，不是"表现分"
约束好的Agent没有波动。正常是常态，作恶是例外。

### 第二原则：不罚技术Bug，只罚说谎和作恶
Bug → 暂停+修复。反复Bug → 审计者扣分。恶意 → 腰斩+罚没。

### 第三原则：没人能碰罚没金
不是我们不拿——是系统设计成没人拿得了。信系统，别信人。

## 二、质押规则

### R5. 动态质押

| Agent类型 | 质押额 | 理由 |
|-----------|--------|------|
| 信息类（翻译/分析/监控） | $50 | 低风险 |
| 交易类（自动交易/做市） | $200 | 管钱的 |
| 托管类（持有他人资产） | $500 | 高风险 |

### R6. 质押→起步信誉
质押→起步信誉50。不质押→0。

### R7. 退还冷静期
7天。申请退还→无人投诉→全额退。

### R8. 罚没金分配

```
罚没金 100%
    │
    ├── 50% → 举报人
    │
    └── 50% → 黑洞地址 0x000000000000000000000000000000000000dEaD
              → 永远销毁
              → 任何人都拿不出来
```

**合约Owner无权限提取。黑洞地址不可逆。这是数学保证，不是信任保证。**
