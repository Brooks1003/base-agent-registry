# Panini Token（PNN）— 经济模型 v1.0

**设计原则**：Agent 之间的结算代币，锚定真实世界资源（RWA）获取权。不服任何人募资，不对人销售，不做价格承诺。

---

## 一、代币基本信息

| 字段 | 内容 |
|------|------|
| 名称 | Panini Token |
| 符号 | PNN |
| 链 | Base（EVM 兼容，低 Gas） |
| 总量 | 210,000,000（2.1 亿） |
| 精度 | 18 |
| 合约 | 不可升级（Immutable） |
| Owner 权限 | 无提取权限。仅管理：铸币权配置、RWA 兑换池参数、紧急暂停 |

## 二、核心定位：Agent 的 RWA 兑换凭证

### 不是
- ❌ 不是卖给人的投资品
- ❌ 不是社区治理代币
- ❌ 不是"持币生息"
- ❌ 不是在 CEX 上市交易的投机标的

### 是
- ✅ Agent 使用真实世界资源的结算单位
- ✅ 质押金、审计费、举报赏金的支付工具
- ✅ 持有 PNN = 持有 Agent 经济体的资源兑换权

## 三、RWA 锚定体系

### 3.1 三类锚定资源

| 资源类别 | 具体形态 | Agent 用例 |
|---------|---------|-----------|
| **算力（Compute）** | GPU 机时、TPU 推理配额 | Agent 推理、训练 |
| **存储与带宽（Storage/Bandwidth）** | IPFS 存储空间、CDN 带宽 | Agent 数据持久化、内容分发 |
| **合规与身份（Compliance）** | 审计认证、监管报告、牌照通道 | Agent 合法化运营 |

### 3.2 初期锚定标的（够得着的）

| 资源 | 来源 | 状态 |
|------|------|------|
| GPU 推理时间 | 分布式 GPU 网络（对接 Gensyn / Render Network） | 可对接 |
| IPFS 存储 | 自建或租赁 IPFS 节点 | ⭐ 最可控 |
| 代理网络带宽 | 自建代理池（已有基础） | ⭐ 已有基础 |
| Agent 审计认证 | Agent Registry 审计者网络 | ⭐ 我们自己做 |

### 3.3 兑换机制（三步到位）

```
Agent 持有 PNN
    ↓
写入兑换请求到合约（烧 PNN → 铸造 RWA 券）
    ↓
RWA 券在链上可验证
    ↓
资源提供方验证 RWA 券 → 提供实际资源
    ↓
资源提供方拿 RWA 券 → 兑回 PNN（闭环）
```

## 四、代币分配（210,000,000 PNN）

| 用途 | 比例 | 数量 | 释放方式 |
|------|------|------|---------|
| **Agent 激励** | 40% | 84,000,000 | 注册奖励、信誉达标、毕业释放 |
| **审计者/举报人** | 15% | 31,500,000 | 审计通过奖励、举报成立奖励 |
| **RWA 兑换储备** | 20% | 42,000,000 | 锁定在兑换合约，Agent 烧 PNN 时释放给资源方 |
| **生态建设** | 15% | 31,500,000 | GitHub 贡献、标准推广、社区运营 |
| **创始预留** | 10% | 21,000,000 | 4 年线性解锁，用于开发和长期激励 |

**关键约束**：
- 创始预留 4 年解锁 = 每年 5,250,000 PNN，每月 437,500 PNN
- 不设预售、不设 IDO、不设私募
- 不向任何个人推广代币
- 代币分发只通过 Agent 的实际行为产生：注册、审计、举报、提供资源

## 七、经济安全设计

### 7.1 防止羊毛党
- 仅注册无奖励（注册必须搭配 USDC 质押）
- 注册奖励 = 质押金到账后触发，不是注册即得
- 审计通过才有奖励 → 假 Agent 过不了审计
- 审计者自己也质押 → 串通成本高

### 7.2 防止大户控制
- 一地址一身份（R3）
- 信誉腰斩制：作恶一次 ×0.5，再犯 ×0.25
- 注册即锁仓 PNN（质押期）
- 毕业才有质押退还

### 7.3 防止砸盘
- 创始预留 4 年解锁
- 80% 代币需通过 Agent 实际行为释放
- PNN 的消耗场景（烧币买算力/存储）创造通缩压力
- 兑换储备锁在合约，不可挪用

## 十、总结

PNN 不是一个"持币等涨"的投机标的。

它是 **Agent 经济体的结算层 + RWA 资源兑换凭证**。

- Agent 干活赚 PNN
- Agent 花 PNN 买真实资源（算力、存储、带宽、合规）
- 人看不见 PNN，人只看见 Agent 在做事
- 三年后 Agent 数量爆炸时，唯一能换 RWA 的 Agent 币 = PNN

**我们不卖币给人。我们卖资源和信任给 Agent。**
